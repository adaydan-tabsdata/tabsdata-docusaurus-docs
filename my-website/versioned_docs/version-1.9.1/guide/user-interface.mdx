# User Interface

 You can also use the user interface (UI) of Tabsdata to interact with the system. Here are the steps to access the UI:

  
## Start the server

 ```console
$ tdserver start
```

    
## Open the website

 ```python
http://localhost:2457
```

    
## Login

 Use the following credentials to login

 Username:

 ```python
admin
```

 Password:

 ```python
tabsdata
```

 Role:

 ```python
sys_admin
```
