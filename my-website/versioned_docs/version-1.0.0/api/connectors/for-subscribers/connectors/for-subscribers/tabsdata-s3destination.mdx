<ParamField path="uri" type="str | List [ str ]" required>
  The URI of the files with format: ‘s3://path/to/files’. It can be a single URI or a list of URIs.
</ParamField>

<ParamField path="credentials" type="dict | S3Credentials" required>
  The credentials required to access the S3 bucket. Can be a dictionary or a S3Credentials object.
</ParamField>

<ParamField path="format" type="str | dict | FileFormat" default="None">
  The format of the file. If not provided, it will be inferred from the file extension of the data. Can be either a string with the format, a FileFormat object or a dictionary with the format as the ‘type’ key and any additional format-specific information. Currently supported formats are ‘csv’, ‘parquet’, ‘ndjson’ and ‘jsonl’.
</ParamField>

<ParamField path="region" type="str" default="None">
  The region where the S3 bucket is located. If not provided, the default AWS region will be used.
</ParamField>
