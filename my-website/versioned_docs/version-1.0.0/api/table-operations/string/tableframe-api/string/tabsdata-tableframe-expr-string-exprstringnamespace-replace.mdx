Replace the first occurence of a pattern with the given string.

**Parameters:**

<ParamField path="pattern" type="str | Expr" required>
  The pattern to replace.
</ParamField>

<ParamField path="value" type="str | Expr" required>
  The value to replace the pattern with.
</ParamField>

<ParamField path="literal" type="bool" default="False">
  Take the pattern as a literal string (not a regex).
</ParamField>

<ParamField path="n" type="int" default="1">
  Number of matches to replace.
</ParamField>

#### Example

```python
>>> import tabsdata as td
>>>
>>> tf: td.TableFrame ...
>>>
>>> tf.select(td.col("a"), td.col("a").str.replace("b", "X").alias("replace"))
>>>

│ b         ┆ X         │

│ null      ┆ null      │
```
