```python
def alphanumeric(
    ascii_only: bool = False,
    *,
    ignore_spaces: bool = False,
) -> SelectorProxy
```

Select all columns whose names contain only letters and digits.

**Parameters:**

<ParamField path="ascii_only" type="bool" default="False">
  Only consider alphabetic the ASCII letters.
</ParamField>

<ParamField path="ignore_spaces" type="bool" default="False">
  Consider only non-white characters in column names.
</ParamField>

#### Example

```python
>>> import tabsdata.tableframe as td_tf
>>>
>>> tf = td_tf.TableFrame({
...     "User01": [101, 102, 103, 104, 105, 106, 107, 108],
...     "Col_2": [8, 7, 6, 5, 4, 3, 2, 1],
...     "Data3": [10, 20, 30, 40, 50, 60, 70, 80],
...     "Label": ["a", "b", "c", "d", "e", "f", "g", "h"]
... })
```

```python
>>> tf.select(td_tf.selectors.alphanumeric())
```
