<summary>cast</summary>


```python
def cast(
    dtype: Boolean | Categorical | Date | Datetime | Decimal | Duration | Enum | Float16 | Float32 | Float64 | Int8 | Int16 | Int64 | Int32 | Int128 | Null | String | Time | UInt8 | UInt16 | UInt32 | UInt64 | type [ Any ],
    *,
    strict: bool = True,
    wrap_numerical: bool = False,
) -> Expr
```

Cast a value to d different type.

**Categories:** type_casting

**Parameters:**

<ParamField path="dtype" type="Boolean | Categorical | Date | Datetime | Decimal | Duration | Enum | Float16 | Float32 | Float64 | Int8 | Int16 | Int64 | Int32 | Int128 | Null | String | Time | UInt8 | UInt16 | UInt32 | UInt64 | type [ Any ]" required>
  The data type to cast to.
</ParamField>

<ParamField path="strict" type="bool" default="True">
  If false, invalid casts produce null’s; if true, an excetion is raised.
</ParamField>

<ParamField path="wrap_numerical" type="bool" default="False">
  If true, overflowing numbers ara handled; if false, an excetion is raised.
</ParamField>

#### Example

```python
>>> import tabsdata as td
>>> import polars as pl
>>>
>>> tf: td.TableFrame ...
>>>
>>> tf = tf.select(td.col("val"), td.col("val").cast(td.Float64)).alias("cast")
>>>
┌──────┬──────┐
│ val  ┆ cast │
│ ---  ┆ ---  │
│ i64  ┆ f64  │
╞══════╪══════╡
│ 1    ┆ 1.0  │
│ 15   ┆ 15.0 │
│ 18   ┆ 18.0 │
│ 60   ┆ 60.0 │
│ 60   ┆ 60.0 │
│ 75   ┆ 75.0 │
│ null ┆ null │
└──────┴──────┘
```

<summary>to_datetime</summary>


```python
def to_datetime(
    format: str | None = None,
    *,
    time_unit: Literal [ 'ns' , 'us' , 'ms' ] | None = None,
    time_zone: str | None = None,
    strict: bool = True,
    ambiguous: Literal [ 'earliest' , 'latest' , 'raise' , 'null' ] | Expr = 'raise',
) -> Expr
```

Convert the string to a datetime.

**Categories:** type_casting

**Parameters:**

<ParamField path="format" type="str | None" default="None">
  The datetime format string (default %Y-%m-%d %H:%M:%S). See [formats](https://docs.rs/chrono/0.4.19/chrono/format/strftime/index.html).
</ParamField>

<ParamField path="time_unit" type="Literal [ 'ns' , 'us' , 'ms' ] | None" default="None">
  \{None, ‘us’, ‘ns’, ‘ms’} If None (default), it inferred from the format string
</ParamField>

<ParamField path="time_zone" type="str | None" default="None">
  Time zone for the resulting value.
</ParamField>

<ParamField path="strict" type="bool" default="True">
  If the conversion fails an error will be raised.
</ParamField>

<ParamField path="ambiguous" type="Literal [ 'earliest' , 'latest' , 'raise' , 'null' ] | Expr" default="'raise'">
  Policy to apply on ambiguos Datetimes: ‘raise’: saises an error ‘earliest’: use the earliest datetime ‘latest’: use the latest datetime ‘null’: set to null
</ParamField>

```python
>>> import tabsdata as td
>>>
>>> tf: td.TableFrame ...
>>>
>>> tf.select(td.col("a"), td.col("a").str.to_datetime().alias("to_datetime"))
>>>
┌─────────────────────┬─────────────────────┐
│ a                   ┆ to_datetime         │
│ ---                 ┆ ---                 │
│ str                 ┆ datetime[μs]        │
╞═════════════════════╪═════════════════════╡
│ 2024-12-13 08:45:34 ┆ 2024-12-13 08:45:34 │
│ 2024-12-15 18:33:00 ┆ 2024-12-15 18:33:00 │
│ null                ┆ null                │
└─────────────────────┴─────────────────────┘
```


</details>
