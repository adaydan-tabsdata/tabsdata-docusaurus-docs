##### `join`

```python
def join(
    other: TdLazyFrame,
    on: str | TdExpr | Sequence [ str | TdExpr ] | None = None,
    how: Literal [ 'inner' , 'left' , 'right' , 'full' , 'semi' , 'anti' , 'cross' , 'outer' ] = 'inner',
    *,
    left_on: str | TdExpr | Sequence [ str | TdExpr ] | None = None,
    right_on: str | TdExpr | Sequence [ str | TdExpr ] | None = None,
    suffix: str = '_right',
    join_nulls: bool = False,
    coalesce: bool | None = None,
) -> TdLazyFrame
```

[View source ↗](https://github.com/tabsdata/tabsdata/tree/main/client/td-sdk/tabsdata/frame.py#L462)

Join the TableFrame with another TableFrame.

**Parameters:**

<ParamField path="other" type="TdLazyFrame" required>
  TheTableFrame to join.
</ParamField>

<ParamField path="on" type="str | TdExpr | Sequence [ str | TdExpr ] | None" default="None">
  Name(s) of the columns to join on. The column name(s) must be in bothTableFrame’s. Don’t use this parameter if using `left_on and right_on parameters, or if how=”cross”.
</ParamField>

<ParamField path="how" type="Literal [ 'inner' , 'left' , 'right' , 'full' , 'semi' , 'anti' , 'cross' , 'outer' ]" default="'inner'">
  Join strategy: * inner: An inner join. * left: A left join. * right: A rigth join. * full: A full join. * cross: The cartesian product. * semi: An inner join but only returning the columns from left TableFrame. *anti*: Rows from the left TableFrame that have no matchin the right TableFrame.
</ParamField>

<ParamField path="left_on" type="str | TdExpr | Sequence [ str | TdExpr ] | None" default="None">
  Name(s) of the columns to join on from the leftTableFrame. It must be used together wit the right_on parameter. It cannot be used with the on parameter.
</ParamField>

<ParamField path="right_on" type="str | TdExpr | Sequence [ str | TdExpr ] | None" default="None">
  Name(s) of the columns to join on from the rightTableFrame. It must be used together wit the left_on parameter. It cannot be used with the on parameter.
</ParamField>

<ParamField path="suffix" type="str" default="'_right'">
  Duplicate columns on the rightTable are appended this suffix.
</ParamField>

<ParamField path="join_nulls" type="bool" default="False">
  Ifnull value matches should produce join rows or not.
</ParamField>

<ParamField path="coalesce" type="bool | None" default="None">
  Collapse join columns into a single column.
</ParamField>

#### Example

```python
>>> import tabsdata as td
>>>
>>> tf1: td.TableFrame ...

│ a    ┆ b    │
│ ---  ┆ ---  │

│ C    ┆ 3    │
│ D    ┆ 5    │

│ M    ┆ 50   │
│ null ┆ 20   │
│ F    ┆ null │
└──────┴──────┘
>>>
>>> tf2: td.TableFrame ...
>>>
┌──────┬──────┐
│ a    ┆ b    │
│ ---  ┆ ---  │
│ str  ┆ i64  │
╞══════╪══════╡
│ A    ┆ 3    │
│ Y    ┆ 4    │
│ Z    ┆ 5    │
│ A    ┆ 0    │
│ M    ┆ 6    │
│ null ┆ 8    │
```
