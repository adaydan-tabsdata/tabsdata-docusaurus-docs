You can use a subscriber to read tables from the Tabsdata server and write them to external systems. Subscribers can write in following file formats: |extensions_list_sub|. Subscribers include built-in support to write tables to following file storage systems:

- [Local File System](local_file_system/main.html)
- [Amazon S3](amazon_s3/main.html)
- [Azure Blob Storage](azure/main.html)

You can also create your own custom connectors to write to other systems.
