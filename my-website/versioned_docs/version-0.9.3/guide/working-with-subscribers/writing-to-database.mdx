You can use a subscriber to write Tabsdata tables to external databases. Subscribers include built-in support to write from the following databases:

- [MySQL](mysql/main.html)
- [MariaDB](mariadb/main.html)
- [Oracle](oracle/main.html)
- [Postgres](postgres/main.html)

You can also create your own custom connectors to write from other systems not listed above.
