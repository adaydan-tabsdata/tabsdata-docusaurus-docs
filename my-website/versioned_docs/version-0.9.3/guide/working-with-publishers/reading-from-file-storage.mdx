title: "Reading from File Storage"
sidebarTitle: "Reading from File Storage"
---

You can use a publisher to read files from external systems and publish them as tables to the Tabsdata server. Publishers can read the following file formats: CSV, jsonl, ndjson, parquet, and log. Publishers include built-in support to read files from the following systems:

- [Local File System](local_file_system/main.html)
- [Amazon S3](amazon_s3/main.html)
- [Azure Blob Storage](azure/main.html)

You can also create your own custom connectors to read from other systems.
