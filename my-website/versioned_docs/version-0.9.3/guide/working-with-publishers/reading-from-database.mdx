sidebarTitle: "Reading from Database"

You can use a publisher to read data from external databases and write to the Tabsdata server. Publishers include built-in support to read from the following databases:

- [MySQL](mysql/main.html)
- [MariaDB](mariadb/main.html)
- [Oracle](oracle/main.html)
- [Postgres](postgres/main.html)

You can also create your own custom connectors to read from other systems not listed above.
