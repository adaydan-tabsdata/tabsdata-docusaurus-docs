Below is the list of destination connectors that are currently supported natively in Tabsdata. Beyond these, you can connect with other external systems as per your need using connector plugins.

**File Storage**

- Amazon S3
- Azure Blob Storage
- Local File System

**Database**

- MySQL
- MariaDB
- Oracle
- Postgres
