Convert a degree value to radians

#### Example

```python
>>> import tabsdata as td
>>>
>>> tf: td.TableFrame ...
>>>
>>> tf = tf.select(td.col("val"), td.col("val").radians().alias("radians"))
>>>

│ val     ┆ radians  │
```
