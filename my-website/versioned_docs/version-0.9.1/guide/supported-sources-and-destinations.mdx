Below is the list of sources and destinations that are currently supported natively in Tabsdata. Beyond these, you can connect with other external systems as per your need using connector plugins.

## Sources#

**File Storage**

- Amazon S3
- Azure Blob Storage
- Local File System

**Database**

- MySQL
- MariaDB
- Oracle
- Postgres

## Destinations#

**File Storage**

- Amazon S3
- Azure Blob Storage
- Local File System

**Database**

- MySQL
- MariaDB
- Oracle
- Postgres
