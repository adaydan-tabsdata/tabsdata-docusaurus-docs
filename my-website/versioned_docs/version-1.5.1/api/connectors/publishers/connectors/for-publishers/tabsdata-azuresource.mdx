Initializes the AzureSource with the given URI and the credentials required to access Azure, and optionally a format and date and time after which the files were modified.

**Parameters:**

<ParamField path="uri" type="str | list [ str ]" required>
  The URI of the files with format: ‘az://path/to/files’. It can be a single URI or a list of URIs.
</ParamField>

<ParamField path="credentials" type="AzureCredentials" required>
  The credentials required to access Azure. Must be an AzureAccountKeyCredentials object.
</ParamField>

<ParamField path="format" type="str | FileFormat" default="None">
  The format of the file. If not provided, it will be inferred from the file extension of the data. Can be either a string with the format or a FileFormat object . Currently supported formats are ‘csv’, ‘parquet’, ‘avro’, ‘ndjson’, ‘jsonl’ and ‘log’.
</ParamField>

<ParamField path="initial_last_modified" type="str | datetime" default="None">
  If provided, only the files modified after this date and time will be considered. The date and time can be provided as a string in [ISO 8601 format](https://en.wikipedia.org/wiki/ISO_8601) or as a datetime object. If no timezone is provided, UTC will be assumed.
</ParamField>
