Initializes the AzureDestination with the given URI and the credentials required to access Azure; and optionally a format.

**Parameters:**

<ParamField path="uri" type="str | list [ str ]" required>
  The URI of the files to export with format: ‘az://path/to/files’. It can be a single URI or a list of URIs.
</ParamField>

<ParamField path="credentials" type="AzureCredentials" required>
  The credentials required to access Azure. Must be an AzureCredentials object.
</ParamField>

<ParamField path="format" type="str | FileFormat" default="None">
  The format of the file. If not provided, it will be inferred from the file extension. Can be either a string with the format or a FileFormat object. Currently supported formats are ‘csv’, ‘parquet’, ‘ndjson’ and ‘jsonl’.
</ParamField>
