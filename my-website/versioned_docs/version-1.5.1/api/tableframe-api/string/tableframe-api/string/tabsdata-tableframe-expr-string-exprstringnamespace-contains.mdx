<summary>contains</summary>


```python
def contains(
    pattern: str | Expr,
    *,
    literal: bool = False,
    strict: bool = True,
) -> Expr
```

Evaluate if the string contains a pattern.

**Parameters:**

<ParamField path="pattern" type="str | Expr" required>
  The pattern to search for.
</ParamField>

<ParamField path="literal" type="bool" default="False">
  Take the pattern as a literal string (not a regex).
</ParamField>

<ParamField path="strict" type="bool" default="True">
  if the given pattern is not valid regex, raise an error.
</ParamField>

#### Example

```python
>>> import tabsdata as td
>>>
>>> tf: td.TableFrame ...
>>>
>>> tf.select(td.col("a"), td.col("a").str.contains("ab").alias("contains"))
>>>
┌──────┬──────────┐
│ a    ┆ contains │
│ ---  ┆ ---      │
│ str  ┆ bool     │
╞══════╪══════════╡
│ a    ┆ false    │
│ ab   ┆ true     │
│ b    ┆ false    │
│ xaby ┆ true     │
│ null ┆ null     │
└──────┴──────────┘
```
