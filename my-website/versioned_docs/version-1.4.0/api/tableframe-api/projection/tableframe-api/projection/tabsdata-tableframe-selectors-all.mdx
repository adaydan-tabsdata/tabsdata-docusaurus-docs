Select all columns in a TableFrame.

#### Example

```python
>>> import tabsdata.tableframe as td_tf
>>>
>>> tf = td_tf.TableFrame({
...     "Id": [1, 2, 3, 4, 5, 6, 7, 8],
...     "Name": ["a", "b", "c", "d", "e", "f", "g", "h"]
... })
```

```python
>>> tf.select(td_tf.selectors.all())
```
