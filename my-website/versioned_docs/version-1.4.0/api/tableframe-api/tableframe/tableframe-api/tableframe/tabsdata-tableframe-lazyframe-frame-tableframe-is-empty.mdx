Checks if a TableFrame has no rows.

#### Example

```python
>>> import tabsdata as td
>>>
>>> tf: td.TableFrame ...
>>>
┌──────┬──────┐
│ a    ┆ b    │

│ str  ┆ i64  │

│ A    ┆ 1    │

>>>

>>>
>>> tf: td.TableFrame ...
>>>
┌──────┬──────┐
│ a    ┆ b    │

│ str  ┆ i64  │
╞══════╪══════╡
└──────┴──────┘
>>>
>>> tf.is_empty()
>>>
True
```
