Replace all null values in the TableFrame with the given value.

**Parameters:**

<ParamField path="value" type="Any | Expr | None" default="None">
  The value to replacenull with.
</ParamField>

#### Example

```python
>>> import tabsdata as td
>>>
>>> tf: td.TableFrame ...
>>>
┌──────┬──────┐
│ x    ┆ y    │
│ ---  ┆ ---  │
│ f64  ┆ f64  │

│ 2.0  ┆ 2.0  │
│ NaN  ┆ NaN  │
│ 4.0  ┆ NaN  │

>>> tf.fill_null(20)

│ x    ┆ y    │

│ 2.0  ┆ 2.0  │
```
