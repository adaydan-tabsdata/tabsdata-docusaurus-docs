  The time format string (default %H:%M:%S)[formats] ([https://docs.rs/chrono/0.4.19/chrono/format/strftime/index.html](https://docs.rs/chrono/0.4.19/chrono/format/strftime/index.html)).

  Whether to parse the date strictly.
</ParamField>

<ParamField path="cache" type="bool" default="True">

</ParamField>
