# Supported Sources and Destinations

 Below is the list of sources and destinations that are currently supported natively in Tabsdata. Beyond these, you can connect with other external systems as per your need using [connecton plugins](../plugins#working-with-connector-plugins).

  
## Sources

 **File Storage**

 - [Amazon S3](../04-working-with-functions/working-with-publishers/reading-from-file-storage/amazon-s3#amazon-s3)
- [Azure Blob Storage](../04-working-with-functions/working-with-publishers/reading-from-file-storage/azure#azure)
- [Local File System](../04-working-with-functions/working-with-publishers/reading-from-file-storage/local-file-system#local-file-system)

 **Database**

 - [MySQL](../04-working-with-functions/working-with-publishers/reading-from-database/mysql#mysql)
- [MariaDB](../04-working-with-functions/working-with-publishers/reading-from-database/mariadb#mariadb)
- [Oracle](../04-working-with-functions/working-with-publishers/reading-from-database/oracle#oracle)
- [Postgres](../04-working-with-functions/working-with-publishers/reading-from-database/postgres#postgres)
