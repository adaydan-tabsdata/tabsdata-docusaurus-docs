Sort the TableFrame by the given column(s) or expression(s).

**Parameters:**

<ParamField path="by" type="td_expr.IntoExpr | Iterable [ td_expr.IntoExpr ]" required>
  Column(s) or expression(s) to sort by.
</ParamField>

<ParamField path="more_by" required>
  Additional colums to sort by.
</ParamField>

<ParamField path="descending" type="bool | Sequence [ bool ]" default="False">
  Specifies if the sorting should be descending or not.
</ParamField>

<ParamField path="nulls_last" type="bool | Sequence [ bool ]" default="False">
  Specifies ifnull values should be placed last.
</ParamField>

<ParamField path="maintain_order" type="bool" default="False">
  Preserve the order of equal rows.
</ParamField>
