Deduplicate rows from the TableFrame.

**Parameters:**

<ParamField path="subset" type="ColumnNameOrSelector | Collection [ ColumnNameOrSelector ] | None" default="None">
  Columns to evaluate for duplicate values. If None, all columns are considered.
</ParamField>

<ParamField path="keep" type="UniqueKeepStrategy" default="'any'">
  Strategy to keep duplicates:first, last, any, none ( eliminate duplicate rows).
</ParamField>

<ParamField path="maintain_order" type="bool" default="False">
  Preserve the order of the rows.
</ParamField>
