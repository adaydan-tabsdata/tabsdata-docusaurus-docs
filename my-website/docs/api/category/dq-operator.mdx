 <a id="tabsdata.dataquality.Filter"></a>
<details open>
<summary>Filter</summary>

```python
class Filter(
    criteria: Criteria,
    to_table: str | None = None,
    include_quality_columns: Literal['none', 'criteria', 'all'] = 'none',
)
```

Bases: `CriteriaOperator`

An operator that filters rows from a table based on data quality criteria.

The filtered rows are removed from the table being processed. They can
either be discarded or redirected to another table.

<a id="tabsdata.dataquality.Filter.include_quality_columns"></a>
<ResponseField name="include_quality_columns">
Returns the data quality columns to included in the table with
filtered out data.
</ResponseField>

<a id="tabsdata.dataquality.Filter.to_table"></a>
<ResponseField name="to_table">
Returns the destination table for filtered rows, or None if they are
discarded.
</ResponseField>

**Categories:** dq-operator

</details>
