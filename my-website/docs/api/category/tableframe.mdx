<summary>assert_has_cols</summary>


```python
def assert_has_cols(
    cols: str | list [ str ],
    exact: bool | None = False,
) -> None
```

[View source ↗](https://github.com/tabsdata/tabsdata/blob/b1.9/client/platform/src/main/python/tabsdata/tableframe/lazyframe/frame.py#L836)

Ensures that the (non-system) columns in the TableFrame match the expected columns.

Raises an exception if the expectation is not met.

If exact is True, the check verifies that the TableFrame contains exactly the expected columns, with no extra or missing ones.

**Categories:** tableframe

**Parameters:**

<ParamField path="cols" type="str | list [ str ]" required>
  The expected column name(s). Can be a string or a list of strings.
</ParamField>

<ParamField path="exact" type="bool | None" default="False">
  If True, checks that the TableFrame contains exactly the specified columns.
</ParamField>

**Raises:**

<ResponseField name="ValueError" type="Exception">
If expected columns are missing or unexpected columns are present in the TableFrame.
</ResponseField>

#### Example

```python
>>> import tabsdata as td
>>>
>>> tf: td.TableFrame ...
>>> tf.assert_has_cols("a")
>>> tf.assert_has_cols(["a", "b"], exact=True)
```

<a id="tabsdata.TableFrame.has_cols"></a>
<details>
<summary>has_cols</summary>


```python
def has_cols(
    cols: str | list [ str ],
    exact: bool | None = False,
) -> Tuple[bool, set[str], set[str]]
```

[View source ↗](https://github.com/tabsdata/tabsdata/blob/b1.9/client/platform/src/main/python/tabsdata/tableframe/lazyframe/frame.py#L755)

Verifies the presence of (non-system) columns in the TableFrame.

If exact is True, the check ensures that the TableFrame contains exactly the specified columns (excluding system columns), with no extras or omissions.

**Categories:** tableframe

**Parameters:**

<ParamField path="cols" type="str | list [ str ]" required>
  The column name(s) to verify. Can be a string or a list of strings.
</ParamField>

<ParamField path="exact" type="bool | None" default="False">
  If True, checks that the TableFrame contains exactly the specified columns.
</ParamField>

**Returns:** A boolean indicating whether the check was successful. A set of columns in cols missing in the TableFrame. A set of columns in the TableFrame missing in cols .

**Return type:** tuple [ bool , set [ str ], set [ str ]]

#### Example

```python
>>> import tabsdata as td
>>>
>>> tf: td.TableFrame ...
>>>
┌──────┬──────┐
│ a    ┆ b    │
│ ---  ┆ ---  │
│ str  ┆ i64  │
╞══════╪══════╡
│ A    ┆ 1    │
└──────┴──────┘
>>>
>>> tf.has_cols("a")
>>>
(True, {}, {"b"})
>>>
>>> tf.has_cols(["a", "c", "d"])
>>>
(False, {"c", "d"}, {"b"})
>>>
>>> tf.has_cols("a", exact=True)
>>>
(False, {}, {"b"})
>>>
>>> tf.has_cols(["a", "b"], exact=True)
>>>
(True, {}, {})
```


</details>
