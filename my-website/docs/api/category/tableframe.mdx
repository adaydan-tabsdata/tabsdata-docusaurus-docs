 <details open>
<summary>**`assert_has_cols`**</summary>

**`assert_has_cols(cols: str | list[str], exact: bool | None = False)`**

Ensures that the (non-system) columns in the TableFrame match the expected
columns.

Raises an exception if the expectation is not met.

If exact is True, the check verifies that the TableFrame contains exactly the
expected columns, with no extra or missing ones.

Example:

```text
>>> import tabsdata as td
>>>
>>> tf: td.TableFrame ...
>>> tf.assert_has_cols("a")
>>> tf.assert_has_cols(["a", "b"], exact=True)
```

<ParamField path="cols">
  The expected column name(s). Can be a string or a list of strings.
</ParamField>

<ParamField path="exact">
  If True, checks that the TableFrame contains exactly the specified
columns.
</ParamField>

**Raises:** ValueError вҖ“ If expected columns are missing or unexpected columns are
    present in the TableFrame.

</details>

<summary>**`has_cols`**</summary>

**`has_cols(cols: str | list[str], exact: bool | None = False)`**

Verifies the presence of (non-system) columns in the TableFrame.

If exact is True, the check ensures that the TableFrame contains exactly the
specified columns (excluding system columns), with no extras or omissions.

Example:

```text
>>> import tabsdata as td
>>>
>>> tf: td.TableFrame ...
>>>
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ a    в”Ҷ b    в”Ӯ
в”Ӯ ---  в”Ҷ ---  в”Ӯ
в”Ӯ str  в”Ҷ i64  в”Ӯ
в•һв•җв•җв•җв•җв•җв•җв•Әв•җв•җв•җв•җв•җв•җв•Ў
в”Ӯ A    в”Ҷ 1    в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”Җв”ҳ
>>>
>>> tf.has_cols("a")
>>>
(True, {}, {"b"})
>>>
>>> tf.has_cols(["a", "c", "d"])
>>>
(False, {"c", "d"}, {"b"})
>>>
>>> tf.has_cols("a", exact=True)
>>>
(False, {}, {"b"})
>>>
>>> tf.has_cols(["a", "b"], exact=True)
>>>
(True, {}, {})
```

<ParamField path="cols">
  The column name(s) to verify. Can be a string or a list of strings.
</ParamField>

<ParamField path="exact">
  If True, checks that the TableFrame contains exactly the specified
columns.
</ParamField>

**Returns:** A boolean indicating whether the check was successful.
A set of columns in cols missing in the TableFrame.
A set of columns in the TableFrame missing in cols.

**Return type:** tuple[bool, set[str], set[str]]

</details>
