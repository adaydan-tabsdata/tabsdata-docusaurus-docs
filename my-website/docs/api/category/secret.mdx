# Secrets

Secrets class representing a secret obtained from an environment variable in the
server.

<details open>

Get the secret value pointed at by the secret. To be used only during execution

</ParamField>

<ParamField path="vault" type="str">
  If multiple vaults exist in the system, the name of the vault to
use. When executing in the server, the URL and token associated to that

numbers and underscores, and can’t begin with a number. Defaults to
“HASHICORP”.
