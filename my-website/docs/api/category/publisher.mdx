# Publisher

 <details open>
<summary>

**`publisher(source: AzureSource | LocalFileSource | MariaDBSource | MySQLSource | OracleSource | PostgresSource | S3Source | SourcePlugin, tables: TableOutput | str | List[str], name: str = None, trigger_by: PublisherTriggerBySpec = None, on_tables: OnTablesSpec = None)`**

</summary>

Decorator to define a function to publish data to Tabsdata.

<ParamField path="source">
  Where to obtain the data that will be provided as an input to the
function.
</ParamField>

<ParamField path="tables">
  Where to store the output of the function.
</ParamField>

<ParamField path="name">
  The name with which the function will be registered.
If not provided, the current function name will be used.
</ParamField>

<ParamField path="trigger_by">
  The trigger that will cause
the function to execute. It can be a table in the system, a list of
tables, None (in which case it must be triggered manually), or a
CronTrigger.
</ParamField>

<ParamField path="on_tables" type="OnTablesSpec">
  Actions
to perform on output tables after executing the function. Can be a
single action or a list of them.
Defaults to None.
</ParamField>

**Returns:** The function converted to a Tabsdata Function.

**Return type:** callable

</details>
