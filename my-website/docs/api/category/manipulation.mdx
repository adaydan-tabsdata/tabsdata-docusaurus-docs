```python
def cast(
    dtypes: Mapping [ td_typing.ColumnNameOrSelector | td_typing.DataType , td_typing.DataType ] | td_typing.DataType,
    *,
    strict: bool = True,
) -> TableFrame
```

[View source ↗](https://github.com/tabsdata/tabsdata/blob/b1.9/client/platform/src/main/python/tabsdata/tableframe/lazyframe/frame.py#L958)

Cast columns to a new data type.

**Categories:** manipulation

**Parameters:**

<ParamField path="dtypes" type="Mapping [ td_typing.ColumnNameOrSelector | td_typing.DataType , td_typing.DataType ] | td_typing.DataType" required>
  Mapping of the column name(s) to the new data type(s).
</ParamField>

<ParamField path="strict" type="bool" default="True">
  IfTrue, raises an error if the cast cannot be performed.
</ParamField>

#### Example

```python
>>> import tabsdata as td
>>>
>>> tf: td.TableFrame ...
>>>
┌──────┬──────┐
│ a    ┆ b    │
│ ---  ┆ ---  │
│ str  ┆ i64  │
╞══════╪══════╡
│ A    ┆ 1    │
│ X    ┆ 10   │
│ C    ┆ 3    │
│ D    ┆ 5    │
│ M    ┆ 9    │
│ A    ┆ 100  │
│ M    ┆ 50   │
│ null ┆ 20   │
│ F    ┆ null │
└──────┴──────┘
>>>
>>> tf.cast({"b":td.Float32}).collect()
>>>
┌──────┬───────┐
│ a    ┆ b     │
│ ---  ┆ ---   │
│ str  ┆ f32   │
╞══════╪═══════╡
│ A    ┆ 1.0   │
│ X    ┆ 10.0  │
│ C    ┆ 3.0   │
│ D    ┆ 5.0   │
│ M    ┆ 9.0   │
│ A    ┆ 100.0 │
│ M    ┆ 50.0  │
│ null ┆ 20.0  │
│ F    ┆ null  │
└──────┴───────┘
```
