# Subscriber

 <details open>
<summary>

**`subscriber(tables: TableInput | str | List[str], destination: AzureDestination | LocalFileDestination | MariaDBDestination | MySQLDestination | OracleDestination | PostgresDestination | S3Destination | DestinationPlugin, name: str = None, trigger_by: SubscriberTriggerBySpec = '*')`**

</summary>

Decorator to define a function to subscribe to data present in Tabsdata.

<ParamField path="tables">
  Where to obtain the data that will be provided as an input to the
function.
</ParamField>

<ParamField path="destination">
  Where to store the output of the function.
</ParamField>

<ParamField path="name">
  The name with which the function will be registered.
If not provided, the current function name will be used.
</ParamField>

<ParamField path="trigger_by">
  The trigger that will cause
the function to execute. It can be a table in the system, a list of
tables, None (in which case it must be triggered manually),
or a CronTrigger. Defaults to all dependencies.
</ParamField>

**Returns:** The function converted to a Tabsdata Function.

**Return type:** callable

</details>
