# Filters

 <details open>
<summary>**`extract_as_columns`**</summary>

**`extract_as_columns(offset: int, length: int)`**

Extract a slice of rows from the table as a column-oriented dictionary.

The result is a mapping of column names to lists of values from the selected
rows.

Example:

```text
>>> import tabsdata as td
>>>
>>> tf: td.TableFrame ...
>>>
в”Ңв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ a   в”Ҷ b   в”Ӯ
в”Ӯ --- в”Ҷ --- в”Ӯ
в”Ӯ str в”Ҷ i64 в”Ӯ
в•һв•җв•җв•җв•җв•җв•Әв•җв•җв•җв•җв•җв•Ў
в”Ӯ A   в”Ҷ 1   в”Ӯ
в”Ӯ X   в”Ҷ 10  в”Ӯ
в”Ӯ C   в”Ҷ 3   в”Ӯ
в”Ӯ D   в”Ҷ 5   в”Ӯ
в”Ӯ M   в”Ҷ 9   в”Ӯ
в””в”Җв”Җв”Җв”Җв”Җв”ҙв”Җв”Җв”Җв”Җв”Җв”ҳ
>>>
>>> tf.extract_as_columns(offset=0, length=2)
    {
        "a": ["A", 1],
        "b": ["X", 10]
    }
```

<ParamField path="offset" type="int">
  The starting row index of the slice.
</ParamField>

<ParamField path="length" type="int">
  The number of rows to include in the slice.
</ParamField>

**Returns:** A dictionary where each key is a column name,
and its value is a list of values from the selected slice.

**Return type:** dict[str, list[Any]]

</details>

<summary>**`extract_as_rows`**</summary>

<summary>**`filter`**</summary>

**`filter(*predicates: Expr | Series | str | Iterable[Expr | Series | str])`**

Apply a filter predicate to an expression.

Elements for which the predicate does not evaluate to true are discarded,
evaluations to null are also discarded.

Useful in an aggregation expression.

Example:

```text
>>> import tabsdata as td
>>>
>>> tf: td.TableFrame ...
>>>
в”Ңв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”¬в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”җ
в”Ӯ state в”Ҷ tickets в”Ӯ
в”Ӯ ---   в”Ҷ ---     в”Ӯ
в”Ӯ str   в”Ҷ i64     в”Ӯ
в•һв•җв•җв•җв•җв•җв•җв•җв•Әв•җв•җв•җв•җв•җв•җв•җв•җв•җв•Ў
в”Ӯ CA    в”Ҷ 1       в”Ӯ
в”Ӯ AL    в”Ҷ 3       в”Ӯ
в”Ӯ CA    в”Ҷ 2       в”Ӯ
```
