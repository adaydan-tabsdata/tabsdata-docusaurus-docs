# Transformer

 <details open>
<summary>**`transformer`**</summary>

**`transformer(input_tables: TableInput | str | List[str], output_tables: TableOutput | str | List[str], name: str = None, trigger_by: TransformerTriggerBySpec = '*', on_output_tables: OnTablesSpec = None)`**

Decorator to define a function to transform data present in Tabsdata.

<ParamField path="input_tables" type="TableInput">
  Where to obtain the data that
will be provided as an input to the function.
</ParamField>

<ParamField path="output_tables" type="TableOutput">
  Where to store the
output of the function.
</ParamField>

<ParamField path="name" type="str">
  The name with which the function will be registered.
If not provided, the current function name will be used.
</ParamField>

<ParamField path="trigger_by" type="TransformerTriggerBySpec">
  The trigger that will cause
the function to execute. It can be a table in the system, a list of
tables, None (in which case it must be triggered manually),
or a CronTrigger. Defaults to all dependencies.
</ParamField>

<ParamField path="on_output_tables" type="OnTablesSpec">
  Actions to perform on output tables after executing the function.
Can be a single action or a list of them.
Defaults to None.
</ParamField>

**Returns:** The function converted to a Tabsdata Function.

**Return type:** callable

</details>
